# InstagramApp  　

# 概要
このアプリはSwiftでInstagramの一部の機能を参考にして作ったSNSアプリです。  
FirebaseのFirestoreやFirestorageのデータベースを使って投稿内容や画像を保存しています。  
サインアップやログイン機能はFirebaseのAuthを使ってユーザー情報を保存しています。  
ActiveLabelを使ってハッシュタグ機能も導入しています。  
  
# 環境  
Swift 5.3.2  
Xcode 12.3
